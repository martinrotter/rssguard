Localization
============
RSS Guard supports localization into [many languages](https://crowdin.com/project/rssguard).

If you are interested in translating RSS Guard, go to the [RSS Guard page on Crowdin](https://crowdin.com/project/rssguard) and check the status of currently supported localizations.

```{warning}
All translators are expected to keep their translations up to date. If a translation is not updated regularly by its author and only a small number of strings is translated, then that translation, along with its team, will eventually be REMOVED from the project. At least 50% of strings must be translated for a translation to be added to the project.
```
