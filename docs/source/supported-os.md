Supported Operating Systems
===========================
RSS Guard is a cross-platform application, and at this point it is known to work on:
* Windows 7+
* GNU/Linux
  * Regular mainstream Linux distributions such as Arch, Ubuntu, Manjaro, Debian, and Fedora
  * PinePhone (smartphone OS)
  * [Inkbox](https://github.com/Szybet/rssguard-inkbox) (e-ink operating system for e-readers)
* BSD (FreeBSD, OpenBSD, NetBSD, etc.)
* macOS 10.15+
* OS/2 (ArcaOS, eComStation)
