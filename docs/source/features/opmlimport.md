OPML File Import
================
RSS Guard can import files in OPML format.

<img alt="alt-img" src="images/opml-import-menu.png" width="600px">

There is a dedicated window where you can specify an OPML file.

When RSS Guard imports the OPML file, it may take quite some time if there are many feeds because online metadata such as website favicons and website titles is fetched as well.

<img alt="alt-img" src="images/opm-import-windows.png" width="600px">

It is possible to specify a category into which feeds should be imported once they are loaded by RSS Guard.
